# GoStack - Gerenciador de Projetos
Desafios da jornada GoStack


Desafios:

1ª Criar um gerenciador de projetos:
    Com esta API, o usuario poderar criar todo um projeto, desde seu nome até as tareferas do seu projeto
    
    Modo de Uso:
    A api, possui 3(três) rotas, que por sua vez permitiram todo o processo de criar e gerenciar os projetos
    
    1ª Rota: app.route("/projects/:id")
    Nesta rota, o usuario poderar atualizar ou deletar o seu projeto, apartir de seu ID
    
    2ª Rota: app.route("/projects")
    Nesta rota, o usuario poderar criar ou listar seu(s) projeto(s)

    3ª Rota: app.route("/projects/:id/task")
    Nesta rota, o usuario poderar adicionar ou deletar as tarefas de seu projetos, informado o seu ID
    

    Como se trata de uma API, o usuario final será o responsavel pela sua interface de requisições!
      
    Você pode usar softwares que ajudam a testar API's, o insomnia é uma otima opção! https://insomnia.rest/
